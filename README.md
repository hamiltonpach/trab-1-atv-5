# trab-1-atv-5

Esse código declara um vetor com 3 elementos, uma matriz 3x3 e um vetor para armazenar o resultado. Primeiro, o programa lê os valores do vetor e, em seguida, lê os valores da matriz.
Depois disso, ele realiza a multiplicação do vetor pelas colunas da matriz. Para cada coluna, o programa multiplica os elementos do vetor pelos elementos correspondentes dessa coluna e soma os resultados, armazenando em um novo vetor.
Por fim, o programa imprime o vetor resultante, que representa o resultado dessa multiplicação.
